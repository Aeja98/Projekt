# Paws & Places

https://pawsplaces.netlify.app/

Paws & Places är en frontend-baserad webbapplikation där användaren kan klicka på ett land i en interaktiv världskarta och upptäcka hundraser som har sitt ursprung i det landet.

## Syfte

Syftet med webbplatsen är att presentera information om hundrasers ursprung på ett visuellt och interaktivt sätt genom att kombinera geografi och hunddata.

Applikationen använder:
- en interaktiv karta för att välja land
- landsdata från en webbtjänst
- hundrasdata från en annan webbtjänst
- JavaScript för att koppla ihop datan och uppdatera gränssnittet dynamiskt

## API:er och bibliotek

### API:er
- **REST Countries API**  
  Används för att hämta information om länder.

- **The Dog API**  
  Används för att hämta information om hundraser och matcha dem mot valt land.

### Bibliotek / plugin
- **Leaflet**  
  Används för att visa och hantera den interaktiva kartan.
